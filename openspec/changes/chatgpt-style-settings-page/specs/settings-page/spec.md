## 修改需求

### 需求:设置页面布局
设置页面必须采用左右分栏布局：左侧垂直导航栏 + 右侧内容区。

#### 场景:桌面端左右分栏
- **当** 屏幕宽度 >= 768px (md 断点)
- **那么** 设置页面必须显示左侧垂直导航栏和右侧内容区，导航栏宽度 180-210px

#### 场景:移动端上下布局
- **当** 屏幕宽度 < 768px
- **那么** 导航栏必须在顶部水平排列，内容区在下方，导航栏可水平滚动

### 需求:导航栏样式
导航项必须使用 `__menu-item` 样式类，带图标和圆角背景高亮。

#### 场景:导航项基础样式
- **当** 渲染导航项
- **那么** 导航项必须显示图标和文字，使用圆角背景

#### 场景:活跃状态高亮
- **当** 用户选中某个导航项
- **那么** 该导航项必须显示高亮背景色（`var(--token-surface-tertiary)`）

### 需求:关闭按钮位置
关闭按钮必须位于左侧导航栏顶部（桌面端）或隐藏在移动端。

#### 场景:桌面端关闭按钮
- **当** 屏幕宽度 >= 768px
- **那么** 关闭按钮必须显示在导航栏顶部

#### 场景:移动端关闭按钮
- **当** 屏幕宽度 < 768px
- **那么** 关闭按钮可以显示在内容区头部

### 需求:容器尺寸
设置页面容器尺寸必须与 ChatGPT 保持一致。

#### 场景:容器最大宽度
- **当** 渲染设置页面
- **那么** 容器最大宽度必须为 680px

#### 场景:容器高度
- **当** 渲染设置页面
- **那么** 容器高度必须为 420px（桌面端），最大高度 85vh

### 需求:ARIA 无障碍属性
设置页面必须实现完整的 ARIA 无障碍属性支持。

#### 场景:对话框角色
- **当** 设置页面打开
- **那么** 容器必须设置 `role="dialog"` 和 `aria-modal="true"`

#### 场景:导航栏角色
- **当** 渲染导航栏
- **那么** 导航栏容器必须设置 `role="tablist"` 和 `aria-orientation="vertical"`

#### 场景:导航项角色
- **当** 渲染导航项
- **那么** 导航项必须设置 `role="tab"` 和 `aria-selected` 属性

#### 场景:内容区角色
- **当** 渲染内容区
- **那么** 内容区必须设置 `role="tabpanel"` 并关联对应导航项

### 需求:导航栏分隔线
导航栏必须与内容区有视觉分隔。

#### 场景:桌面端分隔
- **当** 屏幕宽度 >= 768px
- **那么** 导航栏右侧必须显示 1px 浅色边框

#### 场景:移动端分隔
- **当** 屏幕宽度 < 768px
- **那么** 导航栏底部必须显示 1px 浅色边框

### 需求:容器阴影
设置页面容器必须使用柔和的阴影效果。

#### 场景:阴影样式
- **当** 渲染设置页面
- **那么** 容器必须显示柔和的多层阴影，与 ChatGPT 风格一致
