## 修改需求

### 需求:布局必须使用固定变量系统

布局必须使用 CSS 变量定义关键尺寸，禁止硬编码像素值。

#### 场景:侧边栏宽度
- **当** 用户查看侧边栏
- **那么** 侧边栏宽度为 var(--sidebar-width)（260px）

#### 场景:Header 高度
- **当** 用户查看顶部 Header
- **那么** Header 高度为 var(--header-height)（52px）

#### 场景:输入框最小高度
- **当** 用户查看输入框
- **那么** 输入框最小高度为 var(--composer-min-height)（52px）

### 需求:必须包含 sticky Header 组件

系统必须包含固定在顶部的 Header 组件，高度为 52px。

#### 场景:Header 默认显示
- **当** 用户打开应用
- **那么** 顶部显示固定高度的 Header，背景为纯白色

#### 场景:Header 滚动行为
- **当** 用户滚动内容
- **那么** Header 保持固定在顶部（sticky）

#### 场景:Header 内容结构
- **当** 用户查看 Header
- **那么** 左侧有菜单按钮，中间有会话标题，右侧有操作按钮

### 需求:输入框必须使用 Grid 布局

输入框内部必须使用 CSS Grid 三列布局组织元素。

#### 场景:输入框 Grid 结构
- **当** 用户查看输入框
- **那么** 输入框使用 grid-template-areas: 'leading primary trailing'

#### 场景:左侧按钮区域
- **当** 用户查看输入框左侧
- **那么** 显示一个圆形 + 按钮（占位）

#### 场景:输入区域
- **当** 用户输入消息
- **那么** 输入框在 primary 区域，可自动扩展高度

#### 场景:发送按钮区域
- **当** 用户查看输入框右侧
- **那么** 显示圆形发送按钮

### 需求:输入框必须使用极大圆角和纯色背景

输入框必须使用 28px 圆角，纯白色背景和细边框，禁止使用渐变或阴影装饰。

#### 场景:输入框默认显示
- **当** 用户查看聊天界面
- **那么** 输入框显示为 28px 圆角的纯白背景容器

#### 场景:输入框聚焦效果
- **当** 用户聚焦输入框
- **那么** 边框颜色加深，无阴影或发光效果

#### 场景:输入框阴影
- **当** 用户查看输入框
- **那么** 显示微妙的 shadow-short-composer 阴影

### 需求:侧边栏必须使用完整三段式结构

侧边栏必须包含顶部操作区、中间会话列表、底部区域三部分。

#### 场景:侧边栏默认显示
- **当** 用户查看侧边栏
- **那么** 侧边栏显示为纯白背景，宽度为 260px

#### 场景:顶部区域
- **当** 用户查看侧边栏顶部
- **那么** 显示新建对话按钮（菜单项样式，带图标）

#### 场景:中间会话列表
- **当** 用户查看侧边栏中间
- **那么** 显示可滚动的会话列表

#### 场景:底部区域
- **当** 用户查看侧边栏底部
- **那么** 显示 sticky 定位的底部区域，包含登录/设置入口

### 需求:会话项必须使用菜单项样式

会话项必须使用 group/__menu-item hoverable 样式，禁止使用边框或渐变背景。

#### 场景:会话项默认显示
- **当** 用户浏览会话列表
- **那么** 每个会话项显示图标 + 标题，无边框

#### 场景:会话项 hover 效果
- **当** 用户悬停会话项
- **那么** 显示浅灰背景，右侧出现操作按钮

#### 场景:会话项选中状态
- **当** 用户选中某个会话
- **那么** 该会话项显示高亮背景色，无边框

### 需求:新建按钮必须使用菜单项样式

新建对话按钮必须使用带图标的菜单项样式，禁止使用纯文字链接样式。

#### 场景:新建按钮显示
- **当** 用户查看侧边栏顶部
- **那么** 新建按钮显示为带图标的菜单项样式

#### 场景:新建按钮 hover 效果
- **当** 用户悬停新建按钮
- **那么** 显示浅灰背景

### 需求:背景必须使用纯色无模糊效果

所有背景必须使用纯色，禁止使用 backdrop-filter 玻璃模糊效果。

#### 场景:主区域背景
- **当** 用户查看主内容区
- **那么** 背景显示为浅灰色纯色（#f7f7f7）

#### 场景:无玻璃效果
- **当** 用户使用应用
- **那么** 不显示任何 backdrop-filter 或 blur 效果

### 需求:欢迎界面必须使用大标题居中

欢迎界面必须使用大标题文字居中显示，禁止使用图标。

#### 场景:欢迎语显示
- **当** 用户进入空会话
- **那么** 显示居中的大标题欢迎语（text-2xl font-semibold），无图标装饰

### 需求:设计变量必须使用 token-* 命名系统

CSS 变量必须使用 token-* 命名规范，禁止使用 color-* 命名。

#### 场景:文字颜色变量
- **当** 开发者设置文字颜色
- **那么** 使用 --token-text-primary, --token-text-secondary 等变量

#### 场景:背景颜色变量
- **当** 开发者设置背景颜色
- **那么** 使用 --token-surface-primary 等变量

#### 场景:边框颜色变量
- **当** 开发者设置边框颜色
- **那么** 使用 --token-border-light, --token-border-heavy 等变量
