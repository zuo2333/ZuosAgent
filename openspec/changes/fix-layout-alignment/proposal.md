## 为什么

前端聊天界面的布局存在对齐问题：用户消息气泡无法正确靠右对齐，头像与消息内容的垂直对齐不一致。这些问题导致界面不够专业、不符合 ChatGPT 风格的设计预期。

## 变更内容

- **修复**：用户消息气泡在容器内的右对齐问题
- **修复**：头像与消息气泡的垂直对齐问题
- **优化**：确保所有元素在 ChatGPT 风格下正确响应式布局

## 功能 (Capabilities)

### 新增功能

无新增功能。

### 修改功能

- `chat-message-layout`: 修改消息气泡的布局行为，确保用户消息正确右对齐，头像与内容垂直对齐一致

## 影响

- `frontend/src/components/chat/MessageBubble.tsx` - 消息气泡布局逻辑
- `frontend/src/index.css` - 可能需要调整 CSS 样式
