## 修改需求

### 需求:用户消息右对齐
用户消息气泡必须在消息列表容器内靠右显示，AI 消息必须靠左显示。用户消息气泡最大宽度为 90%，AI 消息最大宽度为 85%。

#### 场景:用户发送消息
- **当** 用户发送一条消息
- **那么** 消息气泡显示在容器右侧，最大宽度不超过 90%

#### 场景:AI 回复消息
- **当** AI 回复消息
- **那么** 消息气泡显示在容器左侧，最大宽度不超过 85%

### 需求:头像与消息顶部对齐
消息头像必须与消息气泡顶部对齐，确保视觉一致性。

#### 场景:短消息对齐
- **当** 消息内容少于一行
- **那么** 头像顶部与气泡顶部对齐

#### 场景:长消息对齐
- **当** 消息内容超过一行
- **那么** 头像顶部与气泡顶部对齐，气泡向下扩展
