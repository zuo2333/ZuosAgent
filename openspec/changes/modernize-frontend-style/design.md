## 上下文

现有前端采用复杂的玻璃拟态设计系统，包含：
- 多层渐变背景 + 3 个浮动光球动画
- 深色侧边栏（#0f172a 渐变玻璃效果）
- 相对较小的圆角（12-16px）
- 紧凑的布局间距

目标参考设计为 ChatGPT 的简约通透风格，特点是干净的背景、浅色通透的侧边栏、更大的圆角和更宽松的留白。

## 目标 / 非目标

**目标：**
- 简化背景，移除装饰性动画
- 将侧边栏从深色改为浅色通透风格
- 增大圆角，统一使用 20-28px 范围
- 增加留白，提升视觉呼吸感
- 保持现有功能完全不变

**非目标：**
- 不添加新功能（如语音输入、模型切换等）
- 不修改后端 API
- 不添加新的依赖库
- 不支持暗色模式（保持浅色主题）

## 决策

### 1. 背景简化

**选择**：采用单色柔和渐变背景，移除浮动光球

**理由**：
- 当前背景过于繁复，分散用户注意力
- 简约背景更符合现代设计趋势
- 减少动画可提升性能和可访问性

**替代方案**：
- 纯白色背景：过于单调，缺乏层次感
- 保持当前设计：视觉过于复杂

### 2. 侧边栏风格

**选择**：浅色通透风格（白色玻璃效果）

**理由**：
- 与主内容区风格统一
- 提升整体通透感
- 符合现代高端设计审美

**替代方案**：
- 保持深色：与简约风格冲突
- 完全透明：缺乏视觉边界

### 3. 圆角系统

**选择**：统一使用更大圆角（20-28px）

| 组件 | 当前值 | 新值 |
|------|--------|------|
| 消息气泡 | 12-16px | 20-24px |
| 输入框 | 16px | 24px |
| 按钮 | 8-12px | 12-16px |
| 卡片 | 16px | 20px |
| 侧边栏会话项 | 12px | 16px |

**理由**：更大圆角营造柔和、友好的视觉感受

### 4. 间距系统

**选择**：整体增加 20-30% 的间距

**理由**：
- 提升视觉呼吸感
- 符合高端简约设计原则
- 改善触控体验

## 风险 / 权衡

| 风险 | 缓解措施 |
|------|----------|
| 用户可能不习惯新风格 | 保持过渡动画，提供视觉连贯性 |
| 浅色侧边栏可能降低对比度 | 使用适当的边框和阴影保持可读性 |
| 移除动画可能影响品牌识别 | 保留核心交互动画（悬停、点击） |
