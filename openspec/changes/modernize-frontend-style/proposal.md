## 为什么

现有前端采用复杂的玻璃拟态设计（多重渐变背景、浮动光球动画、深色侧边栏），视觉上过于繁复。参考 ChatGPT 的简约通透风格，需要将前端改造为更现代、高端、简约通透的设计语言，提升用户体验和视觉舒适度。

## 变更内容

- **背景简化**：移除浮动光球动画，采用柔和的单色渐变或纯色背景
- **侧边栏轻量化**：从深色玻璃效果改为浅色通透风格
- **圆角增大**：统一将圆角从 12-16px 增大到 20-28px
- **留白优化**：增加组件间距和内边距，提升呼吸感
- **动画精简**：保留必要的过渡动画，移除装饰性动画效果
- **色彩统一**：简化配色方案，采用更清爽的色调

## 功能 (Capabilities)

### 新增功能

无新增功能。此变更仅涉及视觉风格改进。

### 修改功能

- `ui-design-system`: 前端设计系统（CSS 变量、组件样式）需要更新为简约通透风格

## 影响

- **代码**：
  - `frontend/src/index.css` - CSS 变量和全局样式
  - `frontend/src/components/layout/` - 布局组件样式
  - `frontend/src/components/chat/` - 聊天相关组件样式
  - `frontend/src/components/session/` - 会话列表组件样式
- **依赖项**：无新增依赖
- **向后兼容**：纯样式变更，不影响功能逻辑
