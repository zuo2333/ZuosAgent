## 为什么

当前 SessionSettings 组件以右侧固定面板形式展示，与主聊天区域并列，占用大量水平空间且视觉风格与总设置（SettingsPage）不一致。用户期望一个更轻量、非阻断式的设置入口，能够在需要时快速调整参数，同时不干扰主对话流程。

## 变更内容

- **修改**: 将 SessionSettings 从右侧固定面板改为右上角弹出面板（popover）形式
- **修改**: 点击设置图标触发弹出，点击面板外部自动关闭
- **修改**: 优化内部 UI 样式，与总设置的视觉风格统一
- **修改**: 固定面板高度（~500px）+ 内部滚动，宽度 ~400px
- **移除**: App.tsx 中右侧滑出面板的布局逻辑

## 功能 (Capabilities)

### 新增功能

- `session-settings-popover`: 会话设置弹出面板，从右上角设置按钮触发，支持点击外部关闭、固定高度滚动、单页分区布局

### 修改功能

无规范级行为变更，仅 UI 展示形式调整。

## 影响

- **前端组件**:
  - `SessionSettings.tsx`: 重构为弹出面板组件，新增 popover 定位逻辑
  - `App.tsx`: 移除右侧面板布局，改为条件渲染弹出面板
  - `ToolSettings.tsx`: 可能需要样式调整以适应新布局
- **用户体验**: 设置入口更加轻量，不干扰主对话区域
