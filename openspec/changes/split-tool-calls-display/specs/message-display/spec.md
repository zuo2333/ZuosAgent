## 新增需求

### 需求:工具调用独立显示

前端渲染 assistant 消息时，每个 `tool_call` 必须作为独立的消息块显示，带有独立的 AI Avatar，而不是合并在一起。

#### 场景:单个工具调用

- **当** assistant 消息包含一个 tool_call
- **那么** 显示一个独立的消息块，包含 AI Avatar 和工具卡片

#### 场景:多个工具调用

- **当** assistant 消息包含多个 tool_calls
- **那么** 每个 tool_call 显示为独立的消息块
- **那么** 每个消息块都有自己的 AI Avatar
- **那么** 消息块之间有适当的间距

#### 场景:工具调用加文本内容

- **当** assistant 消息包含 tool_calls 和文本内容
- **那么** 先显示工具调用消息块，再显示文本消息块
- **那么** 文本消息块也有自己的 AI Avatar
- **那么** 每个消息块独立显示
